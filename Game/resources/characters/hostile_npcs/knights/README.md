# Knight Resources

Knight resources live here so melee and ranged variants can be tuned independently.

- `basic_knight_melee_tuning.tres` owns melee health, speed, detection, attack, respawn, and encounter dialogue defaults.
- `basic_knight_ranged_tuning.tres` owns ranged-specific values and the arrow projectile scene.
- `knight_melee_animations.tres` contains the melee knight animation set.
- `knight_ranged_animations.tres` is a minimal ranged placeholder until the ranged knight animations are ready.
